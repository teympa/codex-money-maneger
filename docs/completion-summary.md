# Smart Kakeibo 完成時点まとめ

## アプリ概要

Smart Kakeibo は、日々の支出記録だけでなく、予算管理、貯金目標、自然文入力、LINE 通知まで一体化した個人用家計簿 Web アプリです。

## できること

- 口座管理
  - 銀行口座、現金、カード、電子マネーを口座として分けて管理
  - 追加、編集、削除

- 明細管理
  - 収入、支出、振替、調整を登録
  - 追加、編集、削除
  - 口座別の流れを維持したまま管理

- 自然文入力
  - 日本語の自然文から明細候補を解析
  - 内容確認後に保存
  - 自動分類ルールと組み合わせて精度向上

- 予算管理
  - 月予算
  - カテゴリ別予算
  - 消化率、残額、予算超過アラート

- 貯金目標
  - 目標金額、現在額、期限、必要月額を管理
  - 追加、編集、削除

- ダッシュボード
  - 今月支出
  - 残予算
  - 今日あと使える額
  - 月末着地予測
  - 口座残高
  - 支払手段別支出

- LINE 通知
  - テスト送信
  - 日次レポート生成
  - Webhook で userId 取得
  - Supabase Cron / 本番URL 運用の土台

## 本番運用面

- Supabase 本番プロジェクト接続
- Vercel デプロイ
- LINE Messaging API 接続
- 日次レポート API
- Supabase Cron 設定手順
- 本番運用チェック表示

## 現時点の位置づけ

特に、日々の記録、予算確認、自然文入力、LINE 通知という利用頻度の高い機能は揃っています。
