Smart Kakeibo

使い方と機能ガイド

Smart Kakeibo は、日々の明細入力だけでなく、予算確認、貯金目標管理、自然文入力、 LINE 通知までまとめて使える個人用家計簿アプリです。

主な機能

1. 口座管理

銀行、現金、カード、電子マネーを別口座として管理できます。

  • 追加、編集、削除
  • 現在残高の確認
  • 支払手段ごとの整理

2. 明細管理

収入、支出、振替、調整を区別して登録できます。

  • 追加、編集、削除
  • 口座ごとの流れを維持
  • 一覧とダッシュボードへ反映

3. 自然文入力

自然な日本語から明細候補を作り、確認して保存できます。

  • 例: 今日セコマで620円、昼ごはん
  • 自動分類ルールと連携
  • 保存前に修正可能

4. 予算と目標

月予算、カテゴリ予算、貯金目標を管理できます。

  • 消化率、残額、必要月額の表示
  • 予算超過アラートの土台
  • 追加、編集、削除に対応

基本の使い方

  1. ログイン
    ログイン画面からアプリに入り、ダッシュボードを開きます。
  2. 口座を確認
    口座一覧で銀行、現金、カード、電子マネーを確認します。
  3. 明細を登録
    手入力か自然文入力で支出や収入を追加します。
  4. 予算を見る
    予算画面とダッシュボードで今月の残額や消化率を確認します。
  5. 目標を管理
    旅行積立や防衛資金などの目標を作って進捗を見ます。

よく使うページ

ダッシュボード

今月支出、残予算、今日あと使える額、月末着地予測を確認できます。

明細一覧

登録した明細をまとめて見て、編集や削除ができます。

自然文入力

文章で入力した内容を明細候補に変換して、確認後に保存できます。

設定

カテゴリ、分類ルール、LINE 通知、本番運用チェックを管理できます。

LINE 通知について

LINE と連携すると、テスト送信や日次レポート送信ができます。設定ページから Webhook 確認、 userId 取得、通知設定保存、テスト送信まで行えます。

  • LINE にテスト送信
  • 今日のレポートをプレビュー
  • 日次レポートの送信ログ確認
  • Supabase Cron による自動送信の土台
